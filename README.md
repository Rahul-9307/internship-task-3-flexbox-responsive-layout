## Features

- Responsive alignment of content sections
- Use of `justify-content`, `align-items`, and `flex-wrap`
- Mobile-friendly layout without using frameworks
- Clean and well-commented CSS code

## Technologies Used

- CSS3 (Flexbox)

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in any modern web browser
3. Resize the browser window to see the responsive behavior

- Difference between Flexbox and traditional layouts
- Creating responsive designs without media queries
- Improved layout control for modern web pages
